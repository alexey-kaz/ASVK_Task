Программа создает файл формата .xml с указанными параметрами.

Библиотеки : 
random -  стандартная библиотека С++, из нее использую random_device для назначения вероятности элементам массива значений Top_Prog_Workload, чтобы основная программа чаще выдавала результат, отличный от failure
pugixml - сторонняя библиотека для работы с xml файлами

Компиляция : 
g++ main.cpp -o input

Запуск :
./input proc_num(число процессоров) prog_num(число программ) file.xml (название готового файла)
Пример :
./input 16 128 test128.xml

Результатом работы программы должен быть файл с расширением .xml

Быстрее всего программа справляется с созданием тестов, в которых на каждый процессор приходится минимум 8 программ (как того требует задание). Также предусмотрена проверка на среднюю нагрузку процессора (не менее 50%).
