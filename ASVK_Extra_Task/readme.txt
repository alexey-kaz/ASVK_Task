Дополнительное задание 1:

Параллельном алгоритм  выполняет программа main.cpp.
 
Библиотеки, использованные в параллельном алгоритме :
ctime - библиотека С++ для максимально случайных результатов
pugixml - библиотека для работы с xml файлами
omp - библиотека для создания многопоточных приложений на C++

Компиляция параллельного алгоритма :
cd build
cmake ..
make

Запуск дополнительного задания :
./ASVK file.xml (Имя xml файла в build или путь к нему в системе) thread_num (Число потоков)
Пример:
./ASVK test128.xml 4
